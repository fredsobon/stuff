=== cmds : ===

-puppet apply pour tester un manifest / code puppet : 
puppet apply fichier.pp ou path/tochemin/fichier.pp
puppet apply helloworld.pp 
puppet apply /vagrant/manifests/helloworld.pp 

- puppet ressource pour afficher la config puppet pour une ressource donnee 
puppet resource file /tmp/testfile.txt
file { '/tmp/testfile.txt':
  ensure => 'absent',
}
puppet resource user root
user { 'root':
  ensure           => 'present',
  comment          => 'root',
  gid              => '0',
  home             => '/root',
  password         => '$1$v4K9E8Wj$gZIHJ5JtQL5ZGZXeqSSsd0',
  password_max_age => '99999',
  password_min_age => '0',
  shell            => '/bin/bash',
  uid              => '0',
}
