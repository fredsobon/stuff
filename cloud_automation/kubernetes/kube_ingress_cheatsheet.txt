===== cheatsheet / memo cmd ingress ====


https://kubernetes.github.io/ingress-nginx/troubleshooting/


- examiner les pods qui portent le service ingress : 

 kubectl get pods -n  < namespace ingress> 
ex : 
 kubectl get pods -n ingress-nginx
 NAME                                        READY     STATUS    RESTARTS   AGE
 default-http-backend-5fbc94f784-qsqsqsqs       1/1       Running   0          1d
 nginx-ingress-controller-5855467dc7SsSz        1/1       Running   0          5d


- examiner la conf ingress qui dans cet exemple est un nginx : 

kubectl exec -it -n < namespace ingress >  < notre pod ingress >  cat /etc/nginx/nginx.conf
ex : 
kubectl exec -it -n ingress-nginx nginx-ingress-controller-5855467dc7SsSz cat /etc/nginx/nginx.conf
