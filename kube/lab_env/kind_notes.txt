==== notes kind kubernetes in docker ===


= set up : 

https://kind.sigs.k8s.io/docs/user/quick-start/
curl -Lo ./kind https://kind.sigs.k8s.io/dl/v0.9.0/kind-linux-amd64 && chmod +x ./kind &&  sudo mv ./kind /usr/local/bin/kind
